## What you can build

{% tabs %}
{% tab title="🤖 AI Agents" %}
**Build agents through conversation — no code required.**

Describe what you want an agent to do in plain English. WorkflowFiesta configures it, gives it tools, and makes it available instantly.

- Customer support triage that classifies and drafts replies
- Weekly report bots that pull data and email summaries
- Research assistants that browse the web and synthesize findings
- Internal ops agents that read your files and answer questions
{% endtab %}

{% tab title="⚙️ Automated Workflows" %}
**Chain agents, scripts, and API calls into multi-step pipelines.**

Workflows run on a schedule, respond to a webhook, or trigger from a chat message. Each step feeds the next — no glue code needed.

- Monday morning reports delivered to your inbox before you wake up
- Webhook-triggered pipelines that respond to form submissions or Stripe events
- Nightly data syncs between your internal tools
- Multi-agent pipelines where specialists hand off to each other
{% endtab %}

{% tab title="🔌 Local Connections" %}
**Connect to systems behind your firewall.**

Install the WorkflowFiesta Runner on your machine or server. Agents can then read local files, query internal databases, and run tools that live on your own infrastructure.

- Read files from your local drive or network share
- Query internal databases that aren't exposed to the internet
- Run scripts on your own hardware with full OS access
- Access tools and APIs that require VPN or internal network
{% endtab %}

{% tab title="🧩 Model Flexibility" %}
**Bring your own AI model — no lock-in.**

Connect OpenAI, Anthropic, AWS Bedrock, or any compatible provider. Switch models at any time without migrating your agents or rewriting prompts.

- Use GPT-4o for reasoning-heavy tasks
- Use Claude for long-context document work
- Use Bedrock for enterprise compliance requirements
- Switch models per-agent based on cost and capability
{% endtab %}
{% endtabs %}
