## How Guides Work

Each guide gives you:

{% stepper %}
{% step %}
### The agent design

What agents to create, what system prompts to use, and how to wire them together.
{% endstep %}

{% step %}
### The workflow YAML

A ready-to-paste workflow definition you can drop straight into WorkflowFiesta.
{% endstep %}

{% step %}
### The credentials you need

Exactly which API keys and connections to set up before running.
{% endstep %}

{% step %}
### Expected output

What you'll have in your hands when the workflow runs — so you know what to expect.
{% endstep %}
{% endstepper %}
