## How to Use a Guide

Each guide gives you:

1. **What to build** — the agent or workflow and what it does
2. **The conversation prompt** — copy and paste this into WorkflowFiesta to start building
3. **What the platform will ask** — the questions you'll need to answer
4. **What you'll have when done** — the working automation

You don't need to read the full documentation first. Pick a guide, follow the steps, and you'll have something running in minutes.
